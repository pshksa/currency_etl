# Currency Rates ETL Project

## Описание
Извлечение данных о курсах валют из открытого API Центробанка России.
Обработка данных с использованием Pandas.
Загрузка данных в базу данных PostgreSQL.


## Технологии
- Python 3.10
- PostgreSQL
- Pandas
- Requests
- SQLAlchemy
- psycopg2
